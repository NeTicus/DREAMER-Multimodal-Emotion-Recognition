# Emotion-Recognition
This repository supports research on robust multimodal emotion recognition for human-centric AI. It establishes a standardized benchmark for multi-physiological emotion recognition using widely adopted deep learning models. The project focuses on evaluating multimodal fusion approaches across signals such as audio, visual, and physiological data to better understand their contribution to emotion recognition performance. The DREAMER dataset is used as a case study to demonstrate the benchmarking framework and provide a reproducible baseline for future research in real-world HCI systems.
